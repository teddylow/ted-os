# TED OS Widget v2

## Summary

TED OS Widget v2 is a Zoho CRM homepage dashboard for TEDvisory.

It reads live Deals data and transforms the standard Zoho Deals list into a student-first operational dashboard.

## Features

- Student-first cards
- Smart urgency scoring
- Admissions / Offer / CAS / COE / Visa grouping
- Counsellor workload
- Search and filters
- Intake countdown
- Stage age calculation
- Debug log retained

## Deployment

Zip the contents of:

```text
widget/src/
```

Upload the ZIP to Zoho CRM Widgets and set:

```text
index.html
```

as the entry file.
